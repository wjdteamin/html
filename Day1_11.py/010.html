1. 서버 : 사용자가 요청하면 서비스를 제공하는 프로그램 2. 웹서버(http 서버) 사용자가 요청하면 html을 보내주는 서버 프로그램 (우리가 아는 모든 웹사이트는 http서버가 돌린다.) 3. http는 웹에서 사용하는 통신표준 4. http요청을 보내려면 서버의 IP주소와 포트를 알아야한다. 5. 사용자는 아이피(203.253.146.184)를 기억하기 어렵다. 그래서 도메인 주소를 사용한다. 그래서 www.naver.com과 같은 domain name을 사용한다. 6. DNS : 도메인 이름을 IP주소로 바꿔주는 서비스 사용자가 도메인 이름을 입력하면 DNS가 IP로 변환한다.
